# Access OWL/RDF XML (INTERNAL)

Access `x` as a 'pyDOID.owl.xml' by confirming it's class, if already
instantiated, or instantiating the object from a given path.

## Usage

``` r
access_owl_xml(x)
```

## Arguments

- x:

  A 'pyDOID.owl.xml' object or the path to an OWL/RDF XML file that can
  be instantiated as such an object by
  [`owl_xml()`](https://allenbaron.github.io/DO.utils/reference/owl_xml.md).
