# Extract PubMed ID from elink_list object

Extract PubMed ID from an `elink_list` object.

## Usage

``` r
# S3 method for class 'elink_list'
extract_pmid(x, no_result = "warning", ...)
```

## Arguments

- x:

  An `elink_list` object, generated by
  [rentrez::entrez_link](https://docs.ropensci.org/rentrez/reference/entrez_link.html)
  and it's derivatives (e.g.
  [citedby_pmid](https://allenbaron.github.io/DO.utils/reference/citedby_pmid.md))
  when `by_id = TRUE` .

- no_result:

  The type of condition that should be signaled when no PubMed results
  exist in a response; one of "error", "warning" (default), "message" or
  "none".

- ...:

  Additional arguments passed on to
  [`extract_pmid.elink()`](https://allenbaron.github.io/DO.utils/reference/extract_pmid.elink.md).
