# Access DOrepo (INTERNAL)

Access `x` as a 'pyDOID.repo.DOrepo' by confirming it's class, if
already instantiated, or instantiating the object from a given path.

## Usage

``` r
access_DOrepo(x)
```

## Arguments

- x:

  A 'pyDOID.repo.DOrepo' object or the path to a local copy of the
  HumanDiseaseOntology repository.
