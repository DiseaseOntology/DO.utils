# Human Disease Ontology Official Colors

Named, hexadecimal colors for official DO use.

## Usage

``` r
DO_colors
```

## Format

A named, vector of 17 colors in hexadecimal format, including:

- main teal-ish colors: `default`, `mid`, `light`, `website`, `websafe`

- saturated versions (better for plotting): `sat`, `sat_mid`,
  `sat_light`

- older, infrequently used orange colors: `orange`, `orange_mid`,
  `orange_light`

- *NEW* lavender-ish accent colors: `accent1`, `accent1_mid`,
  `accent1_light`

- *NEW* yellow-ish accent colors: `accent2`, `accent2_mid`,
  `accent2_light`
