# Identify Obsolete (INTERNAL)

Identify obsolete terms of the Human Disease Ontology.

## Usage

``` r
identify_obsolete(x, ...)
```

## Arguments

- x:

  An object with information about obsolete terms (use
  [`methods()`](https://rdrr.io/r/utils/methods.html) for available
  classes).

- ...:

  Not used; included for extensibility.
