# Convert PubMed summary to tibble (DEPRECATED)

Convert list result generated by
[`rentrez::entrez_summary()`](https://docs.ropensci.org/rentrez/reference/entrez_summary.html)
from the PubMed database into a tibble.

## Usage

``` r
tidy_pubmed_summary(pm_summary, addl_items = NULL)
```

## Arguments

- pm_summary:

  result from `rentrez::entrez_summary(db = "pubmed", ...)`

- addl_items:

  character vector listing "items" (*i.e.* data variables) to include in
  tidy dataset beyond the default: uid, pubdate, title, authors,
  fulljournalname

## Value

A dataframe with `uid`, `pubdate`, `title`, and `fulljournalname`; may
also include `authors`, `authtype` and `auth_clusterid` if author
information is present and additional variables included with
`addl_items`

## DEPRECATION NOTICE

This function has been in favor of
[`tidy_pub_records()`](https://allenbaron.github.io/DO.utils/reference/tidy_pub_records.md)
and will not work on `esummary_list` objects.
